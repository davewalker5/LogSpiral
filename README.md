# LogSpiral - Computational Shell Morphology

<img src="https://github.com/davewalker5/LogSpiral/blob/main/images/morphospace-explorer.png" width="100%">

This project explores computational shell morphology through mathematical growth models, procedural geometry, quantitative morphological analysis and interactive morphospace exploration.

The work began following a visit to the Oxford University Museum of Natural History and an observation that many shell forms appear to emerge from variations on a surprisingly small set of underlying geometric principles:

> Mollusc shells illustrate beautifully how diversity can arise from just a few variations on one simple mathematical model, encoded in the genes.

The project investigates that idea through a series of computational experiments. Starting from a common growth framework, a range of shell forms can be generated by varying parameters governing growth, coiling, aperture geometry, ornamentation and internal shell structure. Each shell can then be described using a common computational taxonomy and compared quantitatively using a metadata-driven morphological similarity analysis.

The resulting shell library can then be explored interactively using the Morphospace Explorer, which provides data-driven filtering, three-dimensional visualisation and inspection of both external shell morphology and internal structures from pre-generated meshes.

Although highly simplified, the resulting models reproduce many features associated with real molluscan and cephalopod shells, including chambered nautilus-like forms, ammonite-inspired shells and high-spired gastropod morphologies.

The emphasis is not on producing biologically exact simulations. Instead, the project explores how complex and recognisably natural forms can emerge from the interaction of relatively simple developmental rules.

In this sense, the models can be viewed as small experiments in computational natural history, using code not only to generate shell forms, but also to classify and compare them within a shared computational morphospace. They sit somewhere between mathematical visualisation, procedural modelling and developmental morphology, using code as a tool for investigating the relationship between growth process and biological form.

## Current Status

The project currently supports:

- Multiple computational growth geometries
  - Logarithmic spiral growth
  - Centreline-conical growth
  - Conical helix growth
- Internal shell architecture
  - Chamber septa
  - Siphuncle generation
  - Variable shell wall thickness
- Surface morphology
  - Ribbed ornamentation
  - Annulated shell segmentation
  - Pigmentation bands
  - Aperture lip flare
- Rendering
  - Opaque and transparent shell rendering
  - Multiple viewpoints
  - Animated shell growth
  - Contact sheet generation
- Computational morphology
  - Shell classification system
  - Morphological similarity analysis
  - Similarity matrices
  - Hierarchical clustering (dendrograms)
- Interactive exploration
  - Morphospace Explorer
  - Automatic morphology-driven filtering
  - Interactive 3D shell visualisation
  - Semi-transparent shell rendering
  - Pre-generated mesh loading

Future directions include additional shell growth geometries, expanded morphological descriptors, richer similarity metrics, enhanced morphospace exploration and further computational investigations into shell development and form.
